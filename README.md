# 🖐️ Virtual Gesture Calculator

A touchless calculator built using Computer Vision and Hand Gesture Recognition. This project uses Python, OpenCV, MediaPipe, and NumPy to allow users to perform mathematical calculations using hand gestures instead of a physical keyboard or mouse.

---

## 🚀 Features

- Real-time hand tracking using MediaPipe
- Gesture-based virtual button interaction
- Pinch detection for button selection
- Interactive calculator UI with animations
- Supports:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
  - Parentheses ()
  - Decimal values
  - Clear (C)
  - Delete (DEL)
- Live webcam integration
- Modern graphical interface built with OpenCV

## 🎮 How to Use

1. Launch the application.
2. Allow webcam access.
3. Move your hand over calculator buttons.
4. Pinch your thumb and index finger together to select a button.
5. Perform calculations without touching the keyboard or mouse.
6. Press **Q** or **ESC** to exit.
